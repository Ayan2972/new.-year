# new.-year

## Website

A minimal local website has been added to preview the downloaded reel.

- Open `index.html` in a browser or serve the repo root and visit http://localhost:8000
- The page embeds the file located at `downloads/Video by code.craft_21.mp4`.

To run a quick local server:

```bash
python3 -m http.server 8000
# then open http://localhost:8000/index.html
```
